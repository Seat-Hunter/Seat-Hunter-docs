# Seat-Hunter-docs
캡스톤디자인 교과목에서 진행한 AI 기반 발표 훈련 서비스 프로젝트입니다. 프론트엔드와 백엔드 레포지토리를 분리하여 관리하고 있으며, 본 저장소는 프로젝트 소개, 시스템 아키텍처, 기술 문서, 발표 자료 및 기타 프로젝트 산출물을 정리하기 위한 문서 저장소입니다.

# SpeechLab

AI 청중과 함께 발표를 연습하고, 실시간 음성 분석과 돌발 질문을 통해 발표 대응력을 훈련하는 웹 서비스입니다.

발표자는 마이크로 발표를 진행하고, 서비스는 실시간 STT, 발표 속도 분석, 필러 단어 감지, 침묵 감지, AI 돌발 질문, 답변 평가, 꼬리질문, 최종 리포트를 제공합니다.

## 팀 구성

| 이름 | 주요 역할 |
|---|---|
| 신희재 | STT 파이프라인, 발표/답변 모드 구분 및 저장, 질문·답변 기록, 발표 리포트 조회, 히스토리 관리, 세션 상태 관리, DB 설계 |
| 신연주 | STT, 발표 분석(필러/침묵/긴장도/실시간 피드백), LLM 질문 타이밍·생성·평가, 종합 점수, 사용자 인증, DB 관리 |
| 곽채령 | STT, 질문 타이밍·생성, 답변 평가, 꼬리질문 생성, 리포트(종합 점수/WPM/인터럽트/회복/강점/개선/추천) |
| 최하진 | 발표 환경 설정, STT, WPM 측정, 청중 반응(애니메이션/평가 기반), 질문 생성, 웹 디자인 |

## 주요 기능

### 발표 환경 설정
- 발표 유형 선택: 면접, 학술 발표, 학교 발표, 회의
- 청중 유형 및 청중 수 설정
- 난이도 설정
- 돌발 질문 사용 여부 설정
- 발표 제한 시간 설정
- 발표 대본 입력

### 실시간 음성 인식 및 분석
- 마이크 음성 입력 수집 (MediaRecorder API)
- Deepgram 기반 실시간 STT (한국어)
- 발표 텍스트 실시간 표시
- 발표 모드와 답변 모드 분리, 각각 별도 저장
- WPM 측정, 필러 단어 감지, 침묵 구간 감지
- 긴장도 및 hesitation score 분석
- 실시간 발표 지표 및 피드백 제공

### AI 청중 반응
- 끄덕임, 관심, 무관심, 박수, 질문 의사 표현 애니메이션
- 발표 지표 및 답변 평가 결과 기반 반응 생성

### LLM 돌발 질문
- OpenAI 기반 질문 타이밍 판단
- 발표 문맥 기반 자연스러운 돌발 질문 생성 (개념/근거/이유/예시/연결 질문)
- 질문 중복 방지 및 쿨다운 적용
- 질문 수락/스킵 처리
- ElevenLabs 기반 TTS 질문 음성 재생 (실패 시 텍스트로 진행)

### 질문 답변 평가
- 질문 수신 및 답변 시작/완료 처리
- 답변 STT 수집, 질문 TTS 에코 필터링, 무응답 타임아웃 처리
- 답변 점수 평가 및 꼬리질문 생성 (최대 2회)
- 질문 및 답변 기록 저장

### 발표 리포트
- 종합 점수, 평균 WPM, 필러/침묵/인터럽트 횟수, 회복 점수
- 강점 분석, 개선 포인트 분석, 실천 방법 제안, 다음 훈련 추천
- 발표 대본, 돌발 질문/답변 기록, 질문별 답변 점수 조회

### 히스토리 관리
- 사용자별 발표 세션 목록 조회
- 과거 세션 상세, 리포트, 대본, 질문/답변 기록 조회

### 사용자 인증
- 회원가입, 로그인, JWT 인증, 내 정보 조회
